# Indian Startup Funding Analysis (Power BI)

## 📌 Project Overview
This project analyzes the **Indian startup funding ecosystem** using Microsoft Power BI.  
The objective is to transform raw startup funding data into meaningful insights through **interactive dashboards and visualizations**.

The dashboard highlights funding trends across industries, cities, and startups, enabling users to better understand investment patterns in India's startup ecosystem.

---

## 📊 Dashboard Insights

The analysis covers:

- **Total Funding:** $90,405.61M  
- **Total Startups:** 525  
- **Total Startup Cities:** 50  
- **Total Industries:** 405  

### Key Findings
- **Top Funded Industries:** Aerospace, Financial Services, Finance  
- **Top Startup Cities:** Mumbai, New Delhi, Bengaluru  
- **Top Funded Startups:** Vistara, Indiabulls Ventures, Fusion Microfinance, ClearTax, Ninjacart  

The dashboard uses visualizations such as **KPI cards, bar charts, treemaps, and geographic maps** to present insights clearly.
